# Datasets for Testing BBT

This directory contains various datasets used in the testing of the BBT package.

## Contents

- `benchmarking_mol.csv` - Adaptation of the benchmarking results from [Praski, Mateusz, Jakub Adamczyk, and Wojciech Czech. "Benchmarking pretrained molecular embedding models for molecular representation learning." arXiv preprint arXiv:2508.06199 (2025).](https://arxiv.org/pdf/2508.06199?) - [data source](https://github.com/scikit-fingerprints/benchmarking_molecular_models/blob/31779f16c004b3fb8aa555ecceb6b95ca71a1d7d/results/arxiv_preprint_2025_08.csv)
